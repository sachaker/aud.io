# **aud.io**
GAN-derived system for converting accents


<img src="https://github.com/sachaker/aud.io/blob/master/data/img/spectrogram.png" width="50%">
  

## Getting started

### Installation

```$ git clone https://github.com/sachaker/aud.io```

### Run program

```$ pip install -r requirements.txt```

```$ python main.py```
